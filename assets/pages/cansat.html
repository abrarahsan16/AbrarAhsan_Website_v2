Engineering Design Team

The ladder iterative algorithm is the capstone project implemented by myself, Muhammad Shirazi, Rehnuba Fairoj and Parham Habibi.

The project was divided into 3 main parts:

  1. The Graphical User Interface (GUI)
  2. Data Parser and Exporter
  3. Calculation Engine

Everyone worked on each part, allowing everyone to learn about the algorithm and its implementation for each stage.

The final report can be found here! Further details are listed below.
