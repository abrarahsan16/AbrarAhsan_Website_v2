PV Integration System

Power Electronics

Course Project

In recent years, there has been an increased demand for renewable energy sources, of which, solar photovoltaic (PV) energy is the most popular. However, PV arrays produce energy that cannot be directly connected to the existing network since PV panels produce DC voltages and currents and the electrical network is based on AC technology. Furthermore, the output voltage and current of the PV arrays is usually not sufficient for direct connection to existing grids. Therefore, challenges are present for power processing techniques to properly integrate solar PV systems into the electrical grid.

The system implemented here is one of the simplest configuration that allows PV panels to be integrated into an AC grid.

The final report submitted can be found here.

System Requirements

The following is a list of specification satisfied by the system:

  • Vdc1 is a DC voltage whose magnitude can vary between 150V and 250V.
  • Vdc2 is a DC voltage whose magnitude should be 400V
  • Vac is a single-phase, AC voltage, whose RMS value is 120V, operating at a frequency of 60Hz.
  • The rated output power of the inverter, P, is 3kW
  • The inductance, Lf, and its winding resistance, Rf, are 1mH and 5ohm respectively.
  • The DC-link capacitor, C, has a capacitance of 4.7mF.
  • Both converters operate at a switching frequency of 10kHz.
  • The system should not include galvanic isolation.

Design of the System

The system was modeled using simulink. The reference figure is the main image on the top of the webpage.

The system consists of an initial DC/DC block, which is chosen to be a Boost Converter, and a DC/AC block, chosen to be a FullBridge circuit, to supply power to the connected electric grid, simulated here using an AC voltage source.

The system is to operate for 2 separate cases, when Vdc1 = 150V as well as when Vdc1 = 250V. The voltage input into the DC/AC block will remain a consistent Vdc2 = 400V. Both blocks are controlled using signal of 10kHz. The DC/DC block uses a simple On-Off mechanism whereas the DC/AC block uses a varying On-Off cycle, created using a sine-wave and triangle wave comparator. The DC/AC block creates a PWM waveform as the output.

For case 1, the system is to supply 1.7kW power to the grid. For case 2, the system is to supply 170W to the grid.

Case 1

A clearer picture of the case 1 outputs can be found in the report.

Based on the requirements, modulation index and angle was calculated required for the grid to receive 1.7kW at unity power factor, while the input voltage was 150V. After that, the inductance was calculated for the DC/DC converter to ensure peak-to-peak ripple current of the inductor did not exceed more than 10% of the AC-side current's fundamental component. Finally, the conduction mode of the DC/DC converter was decided.

The system was designed with a safety margin of 20% for each component.

Case 2

A clearer picture of the case 2 outputs can be found in the report.

While keeping the same calculated parameters, the duty cycle was adjusted while adjusting the input voltage to 250V.

Once the system is still able to supply the same power ouptut, the modulation index and magnitudes of the fundamentals were calculated to get a 10% decrease in power output while maintaining the 250V input.

Conclusion

Overall, the simulation is able to accomplish the required deliverables, with small, acceptable discrepancies. Overall, the project ended with a 95%, with marks lost in the report.
