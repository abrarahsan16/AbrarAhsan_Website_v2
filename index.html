Computer Vision Research Assistant

May 2020 - Present
Robotics and Computer Vision Lab (RCVL), Ryerson University, Toronto, Canada

RCVL is part of the Mechanical Engineering department at Ryerson University.

My research focus was in computer vision for autonomous robotic navigation, particularly working with a neuromorphic vision sensor called SCAMP-5 which is a focal-plane sensor-process (FPSP) array. With the guidance of my supervisor, Dr Sajad Saeedi, my knowledge in robotic navigation and computer vision has improved greatly.

  • Researched topics surrounding the use of deep learning methods for robotic navigation application using mono camera. Topics focused on supervised learning techniques with features learned from images alone
  • Developed a shallow convolutional neural network (CNN) and implemented on a neuromorphic sensor (SCAMP-5) for navigation in corridor and racetrack settings using a Jetbot chassis. Image data was collected from custom Gazebo worlds, verified by running a turtlebot on other custom worlds before transferring to the real world.
  • Implemented BIT-VO, which is a 6-Degress of Freedom visual odometry algorithm which utilizes the FPSP, on a turtlebot to compare against ORB-SLAM implemented on the Intel D455 RGB-D camera
  • Technical expert in the lab for motion capture using Vicon Tracker
